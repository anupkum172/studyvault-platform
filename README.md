Project Overview
StudyVault is a platform where students can upload, organize, search, and share academic resources such as notes, PDFs, and previous-year question papers.

Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT
Cloud Storage: Cloudinary

Setup Instructions
Frontend
cd frontend
npm install
npm start
Backend
cd backend
npm install
npm run dev
